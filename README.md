# ivue-cli 脚手架

ivue 提供了命令行工具，可以通过以下命令进行安装

```
$ npm install -g ivue-material-cli
```

mac 安装
```
$ sudo npm install -g ivue-material-cli
```

再通过 ivue init 并填入项目基本信息新建一个项目

```
$ ivue init
```
